# Portfólio Wagner Inácio

Portfólio pessoal estático, bilíngue e otimizado para GitHub Pages, com foco em Engenharia de Dados & Analytics.

## Stack

- React
- TypeScript
- Vite
- lucide-react
- GitHub Pages

## Como executar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

O build final fica em `dist/`.

## Deploy no GitHub Pages

O workflow em `.github/workflows/deploy.yml` publica automaticamente o site quando houver push na branch `main`.

No GitHub, configure:

1. Repository Settings
2. Pages
3. Source: GitHub Actions

## Conteúdo

O conteúdo bilíngue fica centralizado em `src/content.ts`, incluindo:

- perfil profissional
- habilidades
- experiências
- projetos selecionados
- formação
- contatos públicos
